初认HTML

  • HTML是超文本标记语言
  • 网页是元素组成的一个html文件

Web标准

  • Web标准 是由W3C组织和其他标准化组织制定的一系列集合
  • Web标准的构成 主要包括结构（HTML）表现（CSS）和行为（javascript）

HTML标签

图像标签

  • 属性
    • src 图片路径
    • alt 文本 替换文本，图像不能显示时用文字替换
    • title 文本 提示文本，鼠标放图片上，显示文字
    • width 像素 设置图像的宽度
    • height 像素 设置图像的高度 （可以等比缩放，宽度高度一般只修改一个）
    • border 像素 设置图像的边框粗细
  • 属性之间没有顺序，但标签名与属性，属性和属性之间要有空格
  • 属性=”属性值”（键值对形式）

路径

  • 绝对路径：目录下的绝对位置，一般从盘符开始or网上的图片的地址（\）
  • 相对路径：图片相对HTML页面位置（/)
    • 同一级路径 <img src="文件名" />
    • 下一级路径 <img src="该级路径/文件名" /> (/)
    • 上一级路径 <img src="../文件名" /> (../)

超链接标签

  • 属性

    • href 用于指定连接目标的url地址，（必须属性）
    • target 用于指定链接页面的打开方式 _self为默认值 _blank为在新窗口中打开

  • 链接分类

    • 外部链接 <a href="http://www.baidu.com" target="_self"> 百度</a>
    • 内部链接 <a href="demo.html" target="_blank">内部链接</a> （相对路径）
    • 空链接 <a href="#">空链接</a>
    • 下载链接 <a href="img.zip">下载文件是.zip或.exe格式</a>
    • 网页元素链接 <a href="http://www.baidu.com"> <img src="img.jpg /></a> (为图片创造超链接)

  • 锚点链接(快速定位到页面中某个位置)

    • 在链接文本的herf 设置为 #名字 的形式
    • 在目标位置标签中添加id属性=名字

表格标签

表格主要作用：显示展示数据

  • 输入table>trm>tdn 然后按tab （即m行n列的表格）
  • 表头单元格标签<th></th>
    • （table head缩写，文字加粗 居中显示，直接把td改为th即可）
  • 表格属性
属性名 值 解释
align left center right 规定表格对其方式
border 1或”” 规定单元格边框 “”无边框
cellpadding 像素值 规定单元边沿与其内容之间的空白，默认1像素
width/height 像素值或百分比 规定表格跨度
  • 表格结构标签
    • <thead></thead>表格头部区域
    • <tbody></tbody>表格主体区域
  • 合并单元格（写在td中）
    • 跨行合并：rowspan=”合并单元格的个数”（最上侧单元格为目标单元格，写合并代码）
    • 跨列合并：colspan=”合并单元格的个数”（最左侧单元格为目标单元格，写合并代码）
    • 之后删除多余单元格

列表标签

列表主要作用：布局

分类：

  • 无序列表
  • 有序列表
  • 自定义列表

无序列表

列表项是并列的，ul中只能放li li中可以放任何元素 样式属性CSS更改

有序列表

ol中只能放li li中可以放任何元素 样式属性CSS更改

自定义列表

dl中只能有dt和dd dt和dd个数无限制 通常一个dt有多个dd修饰

表单标签

完整表单包括表单域 表单控件（表单元素） 和提示信息

<input>用于收集用户信息

type属性值 含义
button 定义可点击按钮（JavaScript启动脚本）
checkbox 定义复选框(有相同name属性值)
file 定义输入字段和”浏览”按钮，供文件上传
hidden 定义隐藏的输入字段
image 定义图像形式的提交按钮
password 定义密码字段，掩码
radio 定义单选按钮
reset 定义重置按钮，可清楚表单所有数据
submit 定义提交按钮，会发表单内容发送给服务器
text 定义单行输入字段，可输入文本，默认20个字符
属性 属性值 描述
name 用户自定义 定义input元素名称（单选按钮必须有相同name才可单选）
value 用户自定义 规定input元素值
checked checked 规定此input首次加载是应当被选中（单选按钮和复选框可设置）
maxlength 正整数 规定输入字段中字符最大长度（text）

特殊字符

空格符 &nbsp;

  • < 小于号 < (less than)> 大于号 > (greater than)
  • & 和号 &amp;
  • ¥ 人民币 &yen;
  • © 版权 &copy;
  • ® 注册商标 &reg;
  • ℃ 摄氏度 &deg;
  • ± 正负号 &plusmn;
  • &times;乘号 &times;
  • ÷ 除号 &divide;
  • ² 平方 &sup2;
  • ³ 立方 &sup3;
