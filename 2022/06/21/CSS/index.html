CSS作用

  • 美化网页、布局页面 是一种标记语言
  • CSS主要设置HTML中文本内容、图片外形、版面布局排版

CSS语法规范

  • 选择器以及一条或多条声明

    • 选择器

      • 基础选择器

        • 标签选择器(某一标签全部更改)

          标签名 {

          属性1：属性值1；

          …

          }

        • 类选择器(可以差异化选择不同标签)

          .类名 {

          属性1：属性值1；

          …

          }

          • 在标签中添加 class=”类名1 类名2… “
          • 不要使用纯数字、中文等命名，尽量使用英文字母来表示，可以用 -
          • 一个标签可以有多各类名，多个类名用空格分开

        • id选择器

          #id名 {

          属性1：属性值1；

          …

          }

          • id选择器以#来定义，HTML元素以id属性来设置
          • id属性只能在每个HTML文档中出现一次

        • 通配符选择器

          * {

          属性1：属性值1；

          …

          }

          • *定义，表示先去页面中所有标签

      • 复合选择器

        • 后代选择器(包含选择器)

          元素1 元素2 元素3 …{

          样式声明

          …

          }

          • 可以选择父元素里面的子元素
          • 元素1父级 元素2子级 元素3是孙级..
          • 元素可以不为标签，任意基础选择器就可以

      • 子选择器

        元素1>元素2 {

        样式声明

        …

        }

        • 必须是亲儿子(第一个儿子），不能是孙子

      • 并集选择器

        元素1,

        元素2,

        …{

        元素样式

        …

        }

        • 可以选择多组标签，同时为其定义相同的样式
        • 通常集体声明
        • 任何选择器都可以作为其元素

      • 伪类选择器

        • 可以向选择器添加特殊效果，链接、结构、表单伪类

        • 链接伪选择器

          • a:link (选择所有未被访问的链接)
          • a:visited (选择所有已被访问的链接)
          • a:hover (选择鼠标指针位于其上的链接)
          • a:active (选择活动链接(鼠标按下未弹起的链接))
            • 按照LVHA的顺序写，不颠倒(love hate)
            • 因为a在浏览器有默认样式，所以要专门给a指定样式
            • 实际开发一般只有link和hover

        • :focus伪类选择器(获取光标在的表单元素)

          • input:focus {

            元素样式

            …

            }

字符属性fonts

font-family:字体；

  • 各种字体用逗号隔开
  • 一般情况，若有空格隔开的多个单词组成的字体用引号
  • 最常见：”Microsoft Yahei”, tahoma, arial, “Hiragino Sans GB”

font-size:大小px；

  • 谷歌默认16px

font-weight:数字;

  • 可以由数字表示加粗或者变细
  • 400是normal 700是bold 范围100-900

font-style:属性值

  • normal 正常
  • italic 斜体

字体复合属性

  • font:font-style font-weight font-size/line-height font-family;(顺序不能变)

文本属性

color:属性值

  • 预定义颜色 red green blue pink
  • 十六进制 #FF0000 #FF6600 #29D794
  • RGB代码 rgb(255,0,0)或rgb(100%,0%,0%)

text-align:属性值;

（水平对齐方式）

  • left
  • right
  • center

text-decoration:属性值;

（装饰文本）

  • underline(下划线)
  • line-throu(删除线)
  • over-line(上划线)
  • none(取消默认)

text-indent:数字px;或数字em;

(文本首行缩进)

  • em是一个相对单位，即当前元素的一个文字大小。
  • 如果当前元素没有设置大小，则会按照父元素的1个文字大小。

line-height:数字px;

(行间距)

  • 行间距包括上间距、下间距和文本高度

CSS三种样式表

内部样式表(嵌入式)

  • 一般放在html中
  • 方便控制整个页面
  • 代码元素方便控制
  • 但是没有进行完全分离

行内样式表(行内式)

  • 在中直接给相应标签内部写style=”…”
  • 可以控制当前标签样式
  • 书写繁琐，没有分离，不要大量使用

外部样式表(链接式)

  • 样式单独写在CSS中，之后把CSS文件引入到html中使用
  • ①新建.CSS
  • ②在HTML中使用标签引入

CSS元素显示模式

  • 可以用网页标签来布局网页
  • 元素显示模式就是标签以什么方式进行显示
  • HTML分为块元素和行内元素两种类型

块元素 display: block

  • 自己独占一行
  • 高度宽度以及内外边距可以控制
  • 宽度默认父级100%
  • 容器盒子 可以放行内元素或块级元素
  • 文字类元素内不能放块级元素

行内元素 display: inline

  • 相邻行内元素在一行，一行可以显示多个
  • 高宽直接设置无效，可间接
  • 默认宽度是本身内容宽度
  • 行内元素只能容纳文本或其他行内元素
  • 链接里面不能再放链接,里面可以放块级元素，但是给转换块级模式最安全

行内块元素 display: inline-block;

  • 同时具有块元素和行内元素的特点
  • 一行可以显示多个行内块元素，但之间会有空白缝隙
  • 默认宽度是它本身内容的宽度(行内元素特点)
  • 高度行高内外边距可以控制(块级元素特点)

元素显示模式转换

  • 转换为块元素 display: block;
  • 转换为行内元素 display: inline;
  • 转换为行内块元素 display: inline-block;
  • 单行文字垂直居中
    • line-height文字行高=盒子高度width

CSS三大特性

层叠性

  • 相同选择器设置相同样式
  • 原则：
    • 样式冲突就近原则
    • 样式不冲突不会重叠

继承性

  • 子标签会继承父标签中某些的样式，可以简化代码
  • 原则：
    • 恰当地使用继承可以简化代码 降低CSS复杂性
    • 子元素可以继承父元素样式（text- font- line- colover)
  • 行高的继承：
    • font: 12px/1.5 (意为行高是12px的1.5倍) Yahei;

优先级

  • 原则：

    • 选择器相同层叠性
    • 选择器不同权重优先值
    • 继承的权重是0

  • 权重叠加：如果是复合选择器，会有权重叠加

CSS属性书写顺序

建议遵从以下顺序

1. 布局定位属性：display / position / float / clear / visibility / overflow

2. 自身属性：width / height / margin / padding / border / background

3.文本属性：color / font / text-decoration / text-align / vertical-align / white-pace / break-word

4. 其他属性(CSS3)：content / cursor / border-radius / box-shadow / text-shadow / background:linear-gradient…

content内容

padding内边距

  • padding-top|bottom|left|right: 数字px;

  • 复合型写法：

    • padding: 5px;代表上下左右都有5像素边距
    • padding: 5px 10px;代表上下内边距5像素 左右10像素
    • padding: 5px 10px 20px;代表上内边距5像素 左右内边距10像素 下内边距20像素
    • padding: 5px 10px 20px 30px;上5 右10 下20 左30 （顺时针）

  • 说明：

    • padding会影响盒子实际大小，当盒子已经有width和height时再加padding会撑大盒子 所以要注意用width和height减去
    • 如果盒子本身没有指定width和height则padding不会撑大盒子

margin外边距

  • margin-left|right|top|bottom: 数字px;

  • 复合型写法与padding一致

  • 可以让块级盒子水平居中

    • 盒子必须指定width
    • 盒子左右的外边距都设置为auto
    • 可以写为margin: 0 auto;
    • 若想行内元素或者行内块元素水平居中则给父元素添加text-align: center;

  • 外边距合并问题：

    • 当上下两个兄弟块元素相遇，若上面的元素有下边距margin-bottom 下面元素有上边距margin-top，则他们之间垂直间距不是二者之和 而是取较大值
    • 解决方案：尽量只给一个盒子添加margin值

  • 嵌套块元素垂直外边距塌陷问题：

    • 对于父子级块元素，父元素有上外边距，子元素也有上外边距，此时父元素会塌陷较大外边距值
    • 解决方案：
      • 可以为父元素定义上边框border-top;
      • 可以为父元素定义内边距
      • 可以为父元素添加overflow: hidden;

  • 清除默认内外边距(网页默认)

    ​ * {

    ​ padding: 0;

    ​ margin: 0;

    ​ }

网页布局

网页布局准则

  • 准则1：多块级元素纵向排列-标准流 多块级元素横向排列-浮动
  • 准则2：先设置盒子大小，再设置盒子位置

网页布局整体思路

  1. 确定页面版心(可视区)，测量可知
  2. 分析页面中行模块，以及每个行模块中的列模块
  3. 一行中的列模块经常浮动布局，先确定每个列的大小，后确定列的位置
  4. 制作HTML布局结构，再样式

传统网页布局的三种方式

标准流（普通流 文档流）

  • 块级元素独占一行
  • 行内元素碰到父级元素自动换行
  • 行内块元素

浮动

  • float属性用于创建浮动框，将其移动到一边，直到左边缘或右边缘触及包含块或另一个浮动框的边缘
  • 可以让多个块级盒子一行没有缝隙排列显示，经常用于横向排列盒子

定位

  • 可以让盒子自由的在某个盒子内移动位置或者固定屏幕中某个位置，并且可以压住其他盒子

浮动

选择器 { float: 属性值；}

属性值

  • none 元素不浮动（默认值）
  • left 元素向左浮动
  • right 元素向右浮动

浮动特性

  • 浮动元素会脱离标准流，漂浮在普通流上面
  • 浮动的盒子不再保留原先的位置
  • 如果多个盒子都设置了浮动，则它们会按照属性值一行内显示并且顶端对其排列
  • 注意：浮动的元素是互相贴靠在一起的（没有缝隙），如果父级宽度装不下这些浮动的盒子，多出的盒子会另起一行
  • 浮动元素具有行内块元素特性
  • 如果块级盒子没有设置宽度，默认宽度和父级一样，但是添加浮动后，大小会随着内容决定
  • 浮动经常和标准流父级搭配使用

注意

  • 浮动和标准流的父盒子搭配
  • 一个元素浮动，理论上其余兄弟元素也要浮动
    • 浮动的盒子只会影响后面的盒子的标准流，不会影响前面标准流
    • 即前面盒子标准流独占一行，中间有个浮动盒子，后面标准流盒子被浮动盒子压住

清除浮动

  • 原因

    • 由于父级盒子还在很多情况下不方便设置高度，但是盒子浮动又不占有位置，因此父级盒子会自动为0px从而影响下面的标准流盒子

    • 浮动元素不再占有文档流的位置，会对后面盒子产生影响

  • 本质

    • 父盒子本身有高度则无需清除浮动
    • 清除浮动后，父级会根据浮动盒子自动检测高度，父级有了高度，就不会影响下面标准流

  • 语法

    • 选择器 { clear: 属性值;}
  • 属性值 描述
    left 不允许左侧有浮动元素
    right 不允许右侧有浮动元素
    both 清除左右两侧浮动的影响

  • 清除浮动方法 – 闭合浮动

额外标签法(隔墙法)，W3C做法

  • 在最后一个浮动盒子后添加一个标签
  • 优点：通俗易懂，书写方便
  • 缺点：添加许多无意义标签，结构化较差

父级添加overflow属性

  • 其属性值设置为hidden, auto或scroll
  • 优点：代码简洁
  • 缺点：无法显示溢出部分

父级添加after属性，单伪元素法

  • 优点：没有增加标签，结构更简单
  • 缺点：照顾低版本浏览器
  • 代表性网站：百度、淘宝、网站

父级添加双伪元素

  • 优点：代码更简洁
  • 缺点：照顾低版本浏览器
  • 代表性网站：小米、腾讯

定位

定位组成

定位 = 定位模式 + 边偏移

定位模式

  • 通过position属性设置
  • 子绝父相
值 语义 说明 是否脱标
static 静态定位 静态定位按照标准流特性摆放位置，没有边偏移，很少用 否
relative 相对定位 元素移动时，相对其原来位置来说；
原来的标准流位置继续占有，后面的盒子仍以标准流位置对待;		 否
absolute 绝对定位 元素移动时，相对于祖先元素位置来说；
三个特点：
如果没有祖先元素或者祖先元素没有定位，则以浏览器定位为准对齐
如果祖先元素有定位，则以祖先元素为参考
绝对定位不再占有原位置(脱标)，不影响其他元素		 是
fixed 固定定位 固定于浏览器可视区位置
特点：
以浏览器可视窗口为参照点移动元素，和父元素没有关系，不随滚动条移动
不占有原先位置(脱标)
固定版心右侧算法: **left: 50%; margin-left: 版心*1/2; **		 是
sticky 粘性定位 相对定位＋固定定位混合
必须添加top bottom left right一个才有效
占有原先位置(相对定位特点)
以浏览器可视窗口为参照点移动元素(固定定位特点)		 否

边偏移

  • 定位盒子移动到最终位置
边偏移属性 示例 描述
top top: 80px; 顶端偏移量
bottom bottom: 80px; 底部偏移量
left left: 80px; 左侧偏移量
right right: 80px; 右侧偏移量

定位叠放次序 z-index

  • z-index来控制盒子前后次序
  • 选择器{ z-index: num;}
  • num默认auto，num越大越近

定位的拓展

  • 绝对定位盒子居中
    • 不可以用margin: 0 auto;
    • 算法：left: 50%; margin-left: -自身大小*1/2
  • 定位特殊特性
    • 行内元素加绝对定位或者固定定位，则可以直接加宽高
    • 块元素加绝对定位或者相对定位，如果不给宽高，则默认内容大小
    • 脱标的元素不会塌陷
    • 绝对定位或者固定定位会完全压住盒子，浮动元素只会压住盒子而不会压住文字(文字环绕)

元素的显示与隐藏

diaplay

  • dispaly: none; 隐藏对象，且不占有原先位置
  • display: block; 显示对象

visibility

  • visibility: visible; 显示
  • visibility: hidden; 隐藏，保留原先位置

overflow

  • 对溢出部分显示和隐藏
  • overflow: visible; 默认显示
  • overflow: hidden; 超出部分隐藏
  • overflow: scroll; 溢出部分显示滚动条， 不溢出也显示
  • overflow: auto; 溢出部分显示滚动条， 不溢出不显示

  • JPG
    • 产品类图片，高清，颜色丰富
  • GIF
    • 图片小动画效果，可以透明背景
  • PNG
    • 透明背胶
  • PSD
    • 可以直接获取文字图片，可以测量大小和距离

PS切图方式

  • 图层切图
    • 图层-右键-快速导出png
    • Ctrl+G新建组导出png
  • 切片切图
    • 文件菜单 -> 导出 -> 存储为Web格式 -> 选择图片格式 -> 存储
  • PS插件切图
    • Cutterman
